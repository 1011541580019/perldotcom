Perl Style: Configuration Files

  • If you need a config file, load it with do.
  • This gives you full Perl power.
        # from PCB 8.16
    $APPDFLT = "/usr/local/share/myprog";
    do "$APPDFLT/sysconfig.pl";
    do "$ENV{HOME}/.myprogrc";
