Perl Style: Reduce Complexity (solution)

  • Write this instead:
        while (C1) {
        unless (C2) {
            statement;
            next;
        }
        statements;
        next unless C3;
        statements;
    }
  • Or perhaps even:
        while (C1) {
        statement, next unless C2;
        statements;
        next unless C3;
        statements;
    }
