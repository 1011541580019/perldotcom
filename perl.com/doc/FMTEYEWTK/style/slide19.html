Perl Style: Use Hashes for the First Time

  • A hash is a good way to keep track of whether you've done something before.
  • Embrace the ... unless $seen{$item}++ notation:
        %seen = ();
    foreach $item (genlist()) {
        func($item) unless $seen{$item}++;
    }
